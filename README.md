# AIDS-Nivel-Educacional
Estudo da relação do nível educacional da população brasileira com a infestação da Aids entre 1980 e 2017

Desenvolvido através de análise de dados com Pandas.
O DataSet utilizado vem do DataSus, na aba de Epidemiologias e Morbidade.
http://www2.datasus.gov.br/DATASUS/index.php?area=0203&id=6930&VObj=http://www2.aids.gov.br/cgi/deftohtm.exe?tabnet/

Ao final, será plotado um gráfico para facilitar a visualização e o entendimento da relação entre essas duas variáveis (AIDS e nível educacional)
